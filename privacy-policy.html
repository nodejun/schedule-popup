ShortScheduler 개인정보처리방침

시행일: 2026년 3월 29일

1. 수집하는 정보

ShortScheduler(이하 "본 확장프로그램")는 다음 정보를 처리합니다:

  • Google Calendar 이벤트 데이터: 사용자가 Google Calendar 연결을 승인한 경우, 이벤트 제목, 시작/종료 시간, 색상 정보를 읽고 씁니다.
  • 로컬 스케줄 데이터: 사용자가 직접 생성한 일정(제목, 시간, 색상, 완료 상태)을 Chrome 브라우저의 로컬 저장소(chrome.storage.sync)에 저장합니다.

본 확장프로그램은 사용자의 개인 식별 정보(이름, 이메일, 전화번호 등)를 별도로 수집하지 않습니다.

2. 정보 사용 목적

수집된 정보는 오직 다음 목적으로만 사용됩니다:

  • YouTube 페이지 내에서 캘린더 일정을 표시하고 관리하기 위해
  • Google Calendar와 양방향 동기화를 제공하기 위해

그 외의 어떠한 목적(광고, 분석, 마케팅 등)으로도 사용되지 않습니다.

3. 정보 공유 및 제3자 제공

본 확장프로그램은 사용자 데이터를:

  • 어떠한 제3자에게도 판매하지 않습니다.
  • 광고 플랫폼, 데이터 브로커, 정보 재판매자에게 전송하지 않습니다.
  • 광고 목적(재타겟팅, 개인맞춤형 광고 포함)으로 사용하지 않습니다.
  • 서비스 제공에 필요하지 않은 목적으로 이전하지 않습니다.

4. Google API 제한된 사용 준수

본 확장프로그램의 Google 사용자 데이터 사용 및 전송은 Google API Services User Data Policy를 준수하며, 제한된 사용 요구사항(Limited Use requirements)을 포함합니다.

5. 데이터 저장 및 보안

  • 서버 전송 없음: 모든 데이터는 사용자의 브라우저 내에서만 처리되며, 외부 서버로 전송되지 않습니다.
  • 로컬 저장: 스케줄 데이터는 chrome.storage.sync를 통해 사용자의 Chrome 프로필에 저장됩니다.
  • 보안 통신: Google Calendar API 호출은 모두 HTTPS를 통해 암호화된 채널로 수행됩니다.
  • 토큰 관리: OAuth 액세스 토큰은 Chrome의 내장 토큰 관리 시스템(chrome.identity)으로 관리되며, 확장프로그램이 직접 저장하지 않습니다.

6. 데이터 보존 및 삭제

  • 로컬 스케줄 데이터는 확장프로그램을 삭제하면 자동으로 삭제됩니다.
  • Google Calendar 데이터는 Google 서버에만 존재하며, 본 확장프로그램에 별도 복사본을 보관하지 않습니다.
  • Google Calendar 연결을 해제하면 캐시된 토큰이 즉시 폐기됩니다.

7. 사용자 권리

  • 언제든지 Google Calendar 연결을 해제할 수 있습니다.
  • 언제든지 확장프로그램을 삭제하여 모든 로컬 데이터를 제거할 수 있습니다.
  • Google 계정 설정에서 본 확장프로그램의 접근 권한을 철회할 수 있습니다.

ShortScheduler Privacy Policy

Effective Date: March 29, 2026

1. Information We Collect

ShortScheduler ("the Extension") processes the following information:

  • Google Calendar Event Data: When the user authorizes Google Calendar access, the Extension reads and writes event titles, start/end times, and color information.
  • Local Schedule Data: User-created schedules (title, time, color, completion status) are stored in the browser's local storage (chrome.storage.sync).

The Extension does not collect personally identifiable information (name, email, phone number, etc.) separately.

2. How We Use Information

Collected information is used solely for the following purposes:

  • To display and manage calendar events within YouTube pages
  • To provide bidirectional synchronization with Google Calendar

Data is not used for any other purpose, including advertising, analytics, or marketing.

3. Information Sharing

The Extension does not:

  • Sell user data to any third party.
  • Transfer data to advertising platforms, data brokers, or information resellers.
  • Use data for advertising purposes, including retargeting or personalized advertising.
  • Transfer data for purposes unrelated to the service.

4. Google API Limited Use Compliance

The Extension's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

5. Data Storage and Security

  • No Server Transmission: All data is processed locally within the user's browser and is never transmitted to external servers.
  • Local Storage: Schedule data is stored via chrome.storage.sync in the user's Chrome profile.
  • Secure Communication: All Google Calendar API calls are made over HTTPS encrypted channels.
  • Token Management: OAuth access tokens are managed by Chrome's built-in token management system (chrome.identity) and are not stored directly by the Extension.

6. Data Retention and Deletion

  • Local schedule data is automatically deleted when the Extension is uninstalled.
  • Google Calendar data exists only on Google's servers; the Extension does not maintain separate copies.
  • Disconnecting Google Calendar immediately revokes cached tokens.

7. User Rights

  • You can disconnect Google Calendar at any time.
  • You can uninstall the Extension at any time to remove all local data.
  • You can revoke the Extension's access permissions in your Google Account settings.
